## Spanish translation guidelines

- Address the learner informally and consistently with "tú"; use forms like "te", "ti", "tu", and "tus". No "vos" or "usted" 
- Write clear, natural, broadly Latin American Spanish for beginners; avoid using strongly regional vocabulary.
- Prefer precise explanations over literal translations.
- Prefer natural gender-neutral wording when referring to people.
- Keep code unchanged, including Godot/GDScript keywords, API names, identifiers, numeric values, strings, punctuation, and indentation.
- Preserve all BBCode tags and attributes: glossary terms, URLs, etc.
- Use consistent technical terms for programming, like "matriz" for arrays, and standard Spanish punctuation, including "¿...?" and "¡...!".
