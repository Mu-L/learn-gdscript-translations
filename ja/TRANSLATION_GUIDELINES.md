## Japanese translation guidelines

- Use "です" and "ます" forms for explanations and polite forms like "してください" or "しましょう" for instructions. Headings, labels (like button labels in the UI), and answer choices may use concise noun or plain forms.
- Write clear, natural Japanese for beginners. Prefer precise explanations over literal translations, but do not add or remove technical meaning.
- Omit the subject when natural; use "あなた" sparingly.
- Keep code unchanged, including Godot/GDScript keywords, API names, identifiers, numeric values, strings, punctuation, and indentation.
- Preserve all BBCode tags and attributes: glossary terms, URLs, etc.
- Use consistent Japanese technical terms for programming, and full-width Japanese punctuation like "。", "、", "？", and "！". Remove unnecessary spaces around Japanese text or inline code.
